✔️ Argentina Programa "YoProgramo". | CESSI - Cámara de la Industria Argentina del Software

#YoProgramo es un curso de programación web full-stack ofrecido por Argentina Programa 4.0. Se enseñan habilidades y herramientas para convertirse en un programador full-stack junior. El curso se realiza en línea y con una combinación de teoría y práctica, y se finaliza con una certificación del Ministerio de Economía y la Cámara de la Industria Argentina del Software. Los módulos incluyen introducción a desarrollo web, front-end estático y dinámico, bases de datos, programación orientada a objetos, back-end Java EE, Java Spring boot, Dev Ops, y prácticas profesionales.

✔️ Argentina Programa "Sé Programar". | CESSI - Cámara de la Industria Argentina del Software

Argentina Programa 4.0 ofrece un curso introductorio en programación, que incluye tres módulos: Fundamentos, Programación Imperativa y Programación con Objetos. Los estudiantes deben completar al menos 200 ejercicios y aprobar un examen avalado por el Ministerio de Economía y CESSI. Se utilizarán lenguajes específicos para aprender cada concepto, incluyendo un lenguaje para aprender a programar, JavaScript para programación imperativa y Ruby para el paradigma de objetos.

✔️ Curso introductorio de Internet, la Web y la Programación | ConoSur.Tech

Curso que aborda conceptos básicos de Internet, la Web y la Programación, incluyendo historia, lenguajes, herramientas y preparación para los diplomados. Contiene temas como la evolución de la Web, clasificación de lenguajes, software, control de versiones y un grupo de estudio. Es una guía para una experiencia óptima en los diplomados.

✔️ Introducción a la Programación | Polo TIC Misiones

En este curso aprendí los conceptos básicos de la programación, incluyendo la lógica, sintaxis, control de flujo, variables, operadores y funciones. Aprendí a resolver problemas y crear programas simples con diferentes herramientas y tecnologías. También aprendí a trabajar con diferentes tipos de datos, hacer pruebas, depurar código y documentar. Este curso me brindó una sólida base en la programación y desarrollé habilidades valiosas en este campo.

✔️ Introducción a los Algoritmos y la Programación | TodoCode

El cual me brindó una sólida comprensión de algoritmos y programación, cubriendo temas como estructuras de datos básicas y algoritmos de búsqueda y ordenamiento. También se introdujeron conceptos avanzados y se practicó la programación a través de proyectos y ejercicios. Este curso proporcionó una sólida base para el aprendizaje continúo en tecnología.
